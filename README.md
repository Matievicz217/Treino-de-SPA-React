# Treino de SPA com React

Este projeto foi criado com o objetivo de praticar os conceitos fundamentais do React, com foco em **Single Page Application (SPA)** e **componentização**.

A ideia principal não é construir uma aplicação complexa, mas sim reforçar a base do React, entendendo como organizar componentes, estruturar páginas e trabalhar a navegação sem recarregar a página.

## Objetivo do projeto

- Praticar a criação de uma aplicação SPA
- Trabalhar com componentes reutilizáveis
- Organizar um projeto React de forma simples e clara
- Reforçar conceitos básicos do React na prática

## Tecnologias utilizadas

- React
- JavaScript
- HTML
- CSS

## Conceitos praticados

- Single Page Application (SPA)
- Componentização
- Organização de pastas
- Estrutura básica de um projeto React
